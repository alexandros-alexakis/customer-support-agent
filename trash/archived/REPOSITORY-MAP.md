# Repository Map

> Archived. Superseded by the new docs/ folder structure.

This file mapped every root-level file and folder. The repo has since been restructured. See the docs/ folder and README.md for the current layout.

## Key folders

- `engine/` - triage engine: classifier, prioritizer, escalation, pipeline
- `rag/` - semantic KB retrieval: sync, retriever, ChromaDB
- `feedback/` - gap tracking and correction loop
- `multilingual/` - language detection and multilingual response
- `integrations/` - Zendesk webhook and API client
- `evaluation/` - test cases, evaluation scripts, report generator
- `tests/` - unit tests
- `knowledge-base/` - policy and FAQ markdown documents
- `docs/` - all documentation (new structure)
- `trash/archived/` - old files archived during restructure
