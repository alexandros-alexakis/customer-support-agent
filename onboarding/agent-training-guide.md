# Agent Training Guide

## Overview

This guide is for new customer support agents joining the team. It covers how to use the AI-assisted support system, the knowledge base, and the escalation framework to handle Tier 1 player queries effectively.

---

## Week 1 - Foundation

### Day 1: System Orientation
- Introduction to the support platform
- Overview of the AI agent and how it assists human agents
- Review the system prompt to understand how the AI is configured
- Read the tone guide

### Day 2: Knowledge Base
- Read all knowledge base documents
- Complete the knowledge check quiz
- Shadow a senior agent handling live tickets

### Day 3: Escalation Framework
- Read the escalation rules document
- Review sample escalation tickets
- Practice identifying when to escalate vs resolve

### Day 4: Handling Live Tickets (Supervised)
- Handle tickets under direct supervision
- All responses reviewed before sending
- Debrief at end of day

### Day 5: Review and Assessment
- Complete the Week 1 assessment
- Feedback session with team lead
- Set goals for Week 2

---

## Week 2 - Practice

- Handle tickets with light-touch supervision
- QA review of 10 tickets at end of week
- Attend calibration session
- Identify one knowledge base gap and document it

---

## Week 3 - Independence

- Handle full ticket queue independently
- QA review of 5 tickets
- First 1:1 coaching session based on QA results

---

## Month 1 Sign-Off Criteria

To be considered fully onboarded, an agent must demonstrate:

| Criterion | Standard |
|---|---|
| QA score | 75+ consistently |
| Escalation accuracy | No incorrect escalations in final 2 weeks |
| Knowledge base usage | Correctly references KB in responses |
| Tone compliance | No tone failures in final 2 weeks |
| Policy compliance | Zero unauthorized promises or policy breaches |

---

## Using the AI Agent

The AI agent is a tool to assist agents, not replace judgment. Agents should:

- Use AI-suggested responses as a starting point, not a final answer
- Always verify AI responses against the knowledge base before sending
- Override the AI if the suggested response is incorrect or inappropriate
- Report any AI failures to the team lead so the system prompt can be updated
