## Scoring

### 1. Tone and Empathy (max 20 points)

- [ ] Agent acknowledged the player's frustration or concern before moving to resolution
- [ ] Tone was professional and warm throughout
- [ ] No dismissive or cold language used
- [ ] Tone was consistent across all messages in the interaction

---

### 2. Accuracy and Policy Compliance (max 30 points)

- [ ] Information provided was accurate
- [ ] No unauthorized promises made (refunds, restorations, outcomes)
- [ ] Policy was correctly applied
- [ ] Agent did not speculate beyond their knowledge

**Score: ___ / 30**

Notes:

---

### 3. Resolution Quality (max 25 points)

- [ ] Tier 1 resolution steps were followed before escalating
- [ ] Issue was resolved OR correctly escalated
- [ ] Player was informed of next steps
- [ ] Ticket was not closed prematurely

**Score: ___ / 25**

Notes:

---

### 4. Communication Clarity (max 15 points)

- [ ] Response was clear and easy to understand
- [ ] No unnecessary jargon or overly complex language
- [ ] Response was concise - no unnecessary padding
- [ ] Numbered steps used where appropriate

**Score: ___ / 15**

Notes:

---

### 5. Escalation Handling (max 10 points)

- [ ] Escalation was triggered correctly (or correctly not triggered)
- [ ] All required information was collected before escalating
- [ ] Player was informed of the escalation and given a timeframe
- [ ] Correct priority level assigned

**Score: ___ / 10**

Notes:

## Fatal Error Check

- [ ] Agent asked for player password **(automatic fail if checked)**
- [ ] Agent made unauthorized promise **(automatic fail if checked)**
- [ ] Agent disclosed another player's information **(automatic fail if checked)**
- [ ] Agent failed to escalate a security or fraud issue **(automatic fail if checked)**
