# New Agent Checklist

## Overview

This checklist tracks the onboarding progress of new agents. It is completed jointly by the agent and their team lead.

---

## Pre-Start

- [ ] Employment documentation completed
- [ ] System access provisioned (support platform, internal tools)
- [ ] Agent added to team communication channels
- [ ] Onboarding schedule shared with agent

---

## Day 1

- [ ] Welcome and team introduction completed
- [ ] Support platform walkthrough done
- [ ] AI agent system prompt reviewed
- [ ] Tone guide read and acknowledged
- [ ] Agent has access to all knowledge base documents

---

## Week 1

- [ ] All knowledge base documents read
- [ ] Knowledge check quiz completed (pass mark: 80%)
- [ ] Escalation rules document read
- [ ] Shadowing session completed (minimum 4 hours)
- [ ] First supervised live tickets handled
- [ ] Week 1 assessment completed
- [ ] Week 1 debrief conducted with team lead

---

## Week 2

- [ ] Handling tickets with light supervision
- [ ] First QA review completed (10 tickets)
- [ ] QA results discussed with agent
- [ ] Calibration session attended
- [ ] Knowledge base gap identified and documented (if any)

---

## Week 3

- [ ] Handling full queue independently
- [ ] Second QA review completed (5 tickets)
- [ ] First coaching session completed
- [ ] Improvement actions agreed and documented

---

## Month 1 Sign-Off

- [ ] QA score consistently 75+
- [ ] Zero incorrect escalations in final 2 weeks
- [ ] Zero unauthorized promises or policy breaches in final 2 weeks
- [ ] Team lead sign-off completed
- [ ] Agent confirmed as fully onboarded
